# capstone-2026-Conner-Thalassic_Depths

Two player horror game that will improve my ability to:
  1. Use two player dynamics. I currently haven't gotten two player compatability set up on any game I've worked on.
  2. Create models in Blender. I have created basic shapes using blender but this will improve my capabilities by a lot.
  3. Create and integrate sound effects. Horror games are very sound intensive so this will help a lot with actually integrating these.
  4. This will help me really learn how to make a good game in a shorter amount of time. I always overestimate the time that I have for a project so every project will help me learn where I need to improve on time management.

This game is a FNAF like game. This means that you are in one place trying to defend yourself from threats that are coming towards you. You will need to work together with a friend that is in another office to handle the threats that are coming towards the both of you. 

You will be placed in an underwater facility with a partner. There are different creatures in the facility and in the ocean around you. You will need to listen to your teammate and assist them, otherwise you will both lose.
